The Problem

Aerial video feeds capture significant background movement, like camera shake and wind, which triggers constant false alarms in standard detection models. Furthermore, running heavy Vision Transformers is too computationally slow for real-time tracking on standard processors.

What I Built

🛰️ Spatio-Temporal Fourier Filter

I separated global context capture from high-frequency noise filtering by creating a hybrid ViT-CNN system. Dynamic environmental disturbances (like swaying trees) are blocked using a custom 2D FFT spectral-filter with a learnable noble filter approach.

📐 Kinematic State Validator

I integrated an Extended Kalman Filter (EKF) validation layer directly into the bounding box trajectory estimation, mathematically verifying drone movements to filter out erratic tracking false alarms.

📊 3D Anomaly Dashboard

I built a custom 3D trajectory mapping dashboard that plots target coordinates and maps anomaly classification scores dynamically in a three-dimensional space.

Results

• Filtered 80% of dynamic false alarms, preventing incorrect tracking triggers in cluttered environments.

• Achieved 89% detection accuracy (AUC) under real-world testing conditions.

• Slashed transformer processing overhead by 58x to allow real-time execution speeds.

• Successfully achieved SOTA results on public datasets (like UIT ADrone and Drone Anomaly).

The Problem

Business teams often rely on spreadsheets for KPI reporting, while traditional LLMs can produce inaccurate calculations, inconsistent insights, or malformed outputs when working with raw business data. This makes executive reporting slow and difficult to trust.

What I Built

📐 Reliable Analytics Engine

I separated mathematical computation from AI reasoning by implementing a deterministic regression engine in Python. Financial calculations are verified locally, while the LLM focuses on generating executive insights and recommendations.

🛠️ Self-Healing AI Pipeline

I developed Codemender, an asynchronous recovery pipeline that automatically detects malformed AI responses, repairs invalid JSON, validates the output, and continues processing without interrupting the user experience.

⚡ High-Performance Architecture

I designed a three-layer caching system (Memory → Redis → PostgreSQL) to reduce response time and improve scalability while maintaining consistent performance.

Results

• Produced mathematically grounded executive reports using deterministic forecasting and AI-assisted analysis.

• Reduced response latency through a multi-level caching architecture.

• Improved application reliability with automatic JSON recovery and schema validation.

• Generated boardroom-ready PDF reports with KPI forecasts, benchmark comparisons, and actionable business recommendations.
